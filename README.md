# Listen
Simple MP3-only Music Player; built with Gotk3.

## Building

### Dependencies:

- GTK 3.6-3.22
- GLib 2.36-2.40
- Cairo 1.10
- libasound2
- librsvg2 (*build*)
- fish (*build*)
- meson (*build*)
- fish (*build*)

this project uses go modules, as such it requires go 1.11 minimum.
